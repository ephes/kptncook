# Contributing

## Agent-assisted maintenance

This project is maintained primarily through coding-agent workflows. If you
open an issue or pull request, the first response, investigation, or proposed
patch may be prepared by an agent under maintainer direction.

The maintainer still makes project decisions, reviews changes, and is
responsible for what is merged. Please write issues and pull requests as you
normally would: clear reproduction steps, expected behavior, actual behavior,
and any relevant logs are especially helpful.

For development setup, quality gates, and release workflow, see the
[Contribute section of README.md](README.md#contribute).
